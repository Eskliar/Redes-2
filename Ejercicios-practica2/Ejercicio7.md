## Indicar las clases de las siguientes direcciones, ¿Cuáles son direcciones privadas?
Para saber si una dirección es privada o no, lo único que importa es si pertenece a alguno de los tres rangos de direcciones privadas definidos por RFC 1918.

**Rangos de IP privadas (RFC 1918)**

*Clase A privada*
**10**.0.0.0   - 10.255.255.255

*Clase B privada*
**172.16**.0.0 - **172.31**.255.255

*Clase C privada*
**192.168**.0.0 - 192.168.255.255

- 172.16.7.12 -> Clase B. Privada.
- 200.5.123.221 -> Clase C. No es privada.  
- 224.0.0.9 -> Clase D. No es privada.
